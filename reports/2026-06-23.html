Google 联合微软、NVIDIA 等巨头发布 ARDS 开放标准，让 AI Agent 自主发现网络资源

IndianWeb2 · 阅读原文

Google 联合微软、GitHub、Hugging Face、NVIDIA、亚马逊、思科、Salesforce 等发布 Agentic Resource Discovery Specification（ARDS），这是一套开放标准，让 AI Agent 能自主发现、理解并调用网络上的 API 和服务，无需人工集成。协议采用 Apache 2.0 许可。

Agent 生态的「HTTP 时刻」——标准先行，碎片化才有望终结。
